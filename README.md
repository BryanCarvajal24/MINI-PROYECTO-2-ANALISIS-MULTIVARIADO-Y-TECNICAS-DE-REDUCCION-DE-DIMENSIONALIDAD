# MINI PROYECTO 2 – ANÁLISIS MULTIVARIADO Y TÉCNICAS DE REDUCCIÓN DE DIMENSIONALIDAD

## Descripción
Este proyecto corresponde al **Mini Proyecto 2** de la asignatura **Estadística y Probabilidad II**, desarrollado en el periodo académico **2025-2** en la **Universidad Autónoma de Occidente**.

El trabajo se enfoca en el **análisis exploratorio de datos**, **tratamiento de valores faltantes**, **reducción de dimensionalidad** y **agrupamiento**, aplicados a conjuntos de datos reales asociados al rendimiento de plataformas computacionales y a registros de hurtos urbanos.

---

## Caso 1: Análisis de Rendimiento de una Plataforma de Inferencia
Se analizan métricas de **rendimiento y uso de recursos** de una plataforma computacional, incluyendo tasas de solicitudes, uso de CPU, GPU y memoria, latencias, colas y fallos del sistema.

### Metodología
- Limpieza de datos y corrección de valores inconsistentes  
- **Imputación múltiple** mediante el método PMM  
- Análisis Exploratorio de Datos (EDA)  
- **Análisis de Componentes Principales (ACP)** para reducción de dimensionalidad  
- Construcción de un **Índice de Rendimiento Escalado**  
- **Clustering** mediante K-means para identificar escenarios de alta carga y operación estable  

### Hallazgos
Se identifican dos grupos principales de comportamiento: uno asociado a **alta carga y riesgo de saturación** y otro a **operación estable**, permitiendo caracterizar estados críticos del sistema.

---

## Caso 2: Análisis de Hurtos en la Ciudad de Cali
Se estudian patrones de hurtos considerando variables espaciales, temporales, socioeconómicas y de modalidad del delito.

### Metodología
- Proceso de **ETL** y tratamiento de valores faltantes  
- Imputación múltiple para variables categóricas nominales y ordinales  
- Análisis Exploratorio de Datos  
- **Análisis de Correspondencias Múltiples (ACM)**  
- **Clustering jerárquico (HCPC)** para identificar tipologías de hurto  

### Hallazgos
Se identifican **tres grupos diferenciados de hurtos**, asociados a distintos tipos de delito, zonas de la ciudad, estratos socioeconómicos y franjas horarias.

---

## Herramientas y Técnicas
- Análisis Exploratorio de Datos (EDA)  
- Imputación múltiple  
- ACP y ACM  
- Clustering (K-means y HCPC)  
- Comparación e interpretación estadística de resultados
